maxbutt
======

SSB client for [erlbutt](https://github.com/cmoid/erlbutt), as an emacs extension.

Note: This code originally was forked from a branch of distel, [otp-26][1].
The erlang folks had changed the term format, so we had to hack distel a bit to
upgrade
